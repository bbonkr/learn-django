# learn-django

장고를 배워요

### 가상환경

가상환경 작성

```bash
$ python -m venv devenv
```

가상환경 활성

```bash
$ ./devenv/Scripts/activate
```

pip 업데이트

```bash
$ python -m pip install --upgrade pip
```

### 패키지 목록

```bash
$ pip freeze > requirements.txt
```

### 패키지 설치

```bash
$ pip install -r ./requirements.txt
```
